Bounds

Our objective will be to code a moveable entity will a collision hitbox.

Destructuring the D2PX object.

let { Game, Scene, Entity, Keyboard } = D2PX

Always remember to destructure otherwise, you wouldn't be able to access some constructors.

Creating a Game instance

let myGame = new Game({height: 300})

The height is for the amount of pixels you want along your y-axis.

Creating an entity

Keyboard.init() let mySquare = new Entity({src: "myImage.png" onupdate: (e) => {

  if ( Keyboard.name("UP")) e.move(0, -1)
  if ( Keyboard.name("DOWN")) e.move(0,1)
  if ( Keyboard.name("LEFT")) e.move(-1, 0)
  if ( Keyboard.name("RIGHT")) e.move(1,0)

}})

Atleast the writer did not feel lazy this time.

Creating the obstacle

let myObstacle = new Entity({size: [20,20], color: "red", bounds: { size: [20,20], category: "solid"}})

Just add the bounds property to your entity with the size and the solid category so the built-in move will block on collision. As you can see, the bound is 20 by 20.

let myScene = new Scene({children: [mySquare, myObstacle]})

Put both entities inside the array.

Adding the scene to the game instance

myGame.setScene(myScene)

Adding it to the game.

Start the game

myGame.play()
