Scene

Learn how to exploit the full potential of the scene object.

All the available options

children, range, camera

children: Array of Enities that you want to render inside the scene.

range: The entities within the range will get updated. Only useful if the game's optimization option is on.

camera: The camera of the scene.

let myScene = new Scene({children: [myEntity, myTree], range: 1000, camera: myCamera})

Don't forget to deconstruct D2PX before using the Scene. Otherwise use D2PXScene.

Some useful properties

actif, sleeping

actif: stores an array of the updating entities inside the scene.

sleeping: stores an array of sleeping entities or rather inactif entities.

ids, classes

ids: Stores all the entities and sub-entities inside the scene with a specified id. Returns one entity upon search

classes: Stores all the entities and sub-entities inside the scene with a specified class. Returns an array of entities.

let myEntity = myScene.ids["yourIdName"]
let myEntities = myScene.classes["yourClassName"]

bounds

Stores all categories of bounds inside an array.
